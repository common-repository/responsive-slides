== Description ==

<h3>Responsive Slideshow Plugin </h3>

Based upon Responsive Slides ( http://responsive-slides.viljamis.com/ ) 

Looking for a simple and fast slideshow ? Well, here it is :) . Watch the video tutorial :)

<p> Video Tutorial: http://www.youtube.com/watch?v=h1e4FULbLa4 </p>

== Installation ==



1. Upload `responsive-slider` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Video Guide: http://www.youtube.com/watch?v=h1e4FULbLa4 



== Screenshots ==

1. Able to browse through various saved slideshow, and view their shortcode
2. Featuring guider, to assist you when you are facing troubles.
3. Video Guide: http://www.youtube.com/watch?v=h1e4FULbLa4
